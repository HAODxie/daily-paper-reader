## 摘要
使用基于非侵入式脑电图（EEG）的脑机接口（BCI）解码连续三维（3D）运动想象（MI）仍然具有挑战性，原因是信号变异性和残差解码误差。深度学习架构，如卷积神经网络-长短期记忆（CNN-LSTM）模型，可以捕获连续运动学解码的时空动态；然而，预测轨迹中仍然存在系统性残差误差。我们提出了一种两阶段解码框架，应用强化学习（RL）对CNN-LSTM解码器（CNN-LSTM-RL）的输出进行残差运动学校正。RL智能体在离线条件下训练，无需直接输入EEG，而是对预测的运动学轨迹进行操作，以优化相对于目标轨迹的运动精度。解码性能通过皮尔逊相关系数（$r$）和均方根误差（RMSE）在$x$、$y$和$z$轴上量化。与单独应用CNN-LSTM相比，CNN-LSTM-RL在2D中将平均相关性从$0.5076$提高到$0.7181$（$p=0.0005$），在VR中从$0.6420$提高到$0.7780$（$p=0.0059$），相对增益分别为41.5%和21.2%。相应地，RMSE从$0.0890$降至$0.0532$（2D，$p<0.0001$），以及从$0.0714$降至$0.0441$（VR，$p<0.0001$），相对降低分别为40.2%和38.2%。这些发现表明，这种可扩展的框架通过离线残差RL校正运动学误差，无需额外的神经数据，从而增强了3D BCI MI解码，推动了神经康复、假肢和虚拟交互的发展。

## Abstract
Decoding continuous three-dimensional (3D) motor imagery (MI) using non-invasive electroencephalography (EEG)-based brain--computer interfaces (BCIs) remains challenging due to signal variability and residual decoding errors. Deep learning architectures such as convolutional neural network--long short-term memory (CNN--LSTM) models can capture spatial and temporal dynamics for continuous kinematic decoding; however, systematic residual errors persist in predicted trajectories. We propose a two-stage decoding framework that applies reinforcement learning (RL) to perform residual kinematic correction on the outputs of a CNN--LSTM decoder (CNN--LSTM--RL). The RL agent is trained offline without direct EEG input and instead operates on predicted kinematic trajectories to optimize movement accuracy relative to target trajectories. Decoding performance was quantified using Pearson correlation coefficients ($r$) and Root Mean Square Errors (RMSE) along the $x, y$, and $z$ axes. Compared to CNN--LSTM applied alone, CNN--LSTM--RL improved the mean correlation from $0.5076$ to $0.7181$ ($p = 0.0005$) in 2D and from $0.6420$ to $0.7780$ ($p = 0.0059$) in VR, with relative gains of $41.5\%$ and $21.2\%$, respectively. Correspondingly, RMSE was reduced from $0.0890$ to $0.0532$ (2D, $p < 0.0001$) and from $0.0714$ to $0.0441$ (VR, $p < 0.0001$), representing relative reductions of $40.2\%$ and $38.2\%$. These findings demonstrate that this scalable framework enhances 3D BCI MI decoding by correcting kinematic errors via offline residual RL without extra neural data, advancing neurorehabilitation, prosthetics, and virtual interaction.

- **整体含义**：提出一种两阶段解码框架，通过离线残差强化学习（RL）对CNN-LSTM解码器的输出进行运动学校正，无需额外神经数据或在线交互，有效弥补解码误差，提升连续三维运动想象BCI的实用性，推动神经康复、假肢控制和虚拟交互领域。

## 2. 论文提出的方法论

- **核心思想**：采用残差强化学习，将问题形式化为部分可观察马尔可夫决策过程（POMDP）。第一阶段，利用预训练的CNN-LSTM解码器从EEG的ERSP特征中提取时空特征，输出连续三维速度。第二阶段，一个离线训练的SAC（Soft Actor-Critic）RL智能体仅基于解码器输出、上一时刻校正速度和时间戳作为观测，输出残差速度，加到解码器输出上得到最终校正速度。
- **关键技术细节**：
  - 观测空间：解码器输出的三维速度 $v^{BCI}_t$，上一时刻校正速度 $v_{t-1}$，归一化时间 $t$（不包含EEG或目标轨迹）。
  - 动作空间：连续残差速度 $v^{res}_t$，经 tanh 变换和维度独立缩放后输出，确保校正幅度可控。
  - 奖励函数：负欧氏距离 $r_t = -d(v_t, v^*_t)$，其中 $v_t = v^{BCI}_t + v^{res}_t$，$v^*_t$ 为目标速度。
  - 训练算法：SAC，同时最大化期望回报和政策熵，使用重参数化技巧和维度特定缩放的对数概率校正。
  - 超参数优化：使用 Optuna 进行200次搜索，以5折交叉验证的皮尔逊相关系数为目标，优化学习率、网络层数、批量大小、折扣因子等。

## 3. 实验设计

- **数据集与场景**：
  - 10名右利手参与者，每人完成10个session，分布在2D屏幕和沉浸式VR两种视觉反馈环境下。
  - 任务：上肢到达/运动想象任务，每次到达4个目标，每session含256个试次，共25,600个试次。
  - EEG使用32通道g.Nautilus系统，仅用17个传感器运动区通道；运动轨迹由腕戴Vive跟踪器记录，采样率60Hz。
  - 数据划分为session1用于校准，session2-5用于评估。
- **基准与对比方法**：
  - **基线**：单独CNN-LSTM解码器。
  - **对比策略**：
    - FDG（固定解码器泛化）：session1训练后固定，测试所有session。
    - SAT（顺序自适应训练）：每个session重新训练，测试下一session。
    - WSR（会话内重建）：训练和测试在同一session，作为性能上限。
    - FRL（固定强化学习）：本文方法，session1训练RL智能体后固定，测试所有session。
- **评估指标**：皮尔逊相关系数（$r$）和均方根误差（RMSE），分别在 $x,y,z$ 轴和整体计算。
- **统计检验**：对每种反馈环境分别进行重复测量ANOVA或Friedman检验，事后采用配对t检验或Wilcoxon符号秩检验，并应用Holm-Bonferroni校正。
