## 摘要
虽然语音为心理健康评估提供了丰富、非侵入性的生物标志物，但临床采用受到不透明模型和潜在人口统计偏差的限制。在这项工作中，我们提出了一个方法论框架，用于评估在扩展的DAIC-WOZ数据集上使用低复杂度机器学习基线（RF、SVM和MLP）进行自动抑郁检测的鲁棒性和可解释性，这些模型旨在减轻过拟合并增强泛化能力，同时结合人类可理解的声学特征（MFCCs、eGeMAPS）。为了平衡准确性与临床信任，我们利用可解释性方法（LIME和SHAP）进行特征选择，并通过统计显著性检验和人口统计公平性分析验证我们的发现，以减轻虚假的、伪影驱动的相关性。实证结果表明，由可解释人工智能（XAI）选择的优化特征子集结合MLP架构实现了82%的最先进测试准确率。最终，这项工作为鲁棒且合乎伦理的辅助技术提供了一个透明框架，可应用于任何其他二分类任务。

## Abstract
While speech provides rich, non-invasive biomarkers for mental-health assessment, clinical adoption is limited by opaque models and potential demographic bias. In this work we propose a methodological framework to evaluate robustness and interpretability for automated depression detection on the extended DAIC-WOZ dataset using low-complexity machine learning baselines (RF, SVM, and MLP) chosen to mitigate overfitting and enhance generalization in combination with human-understandable acoustic features (MFCCs, eGeMAPS). To balance accuracy with clinical trust, we leverage explainability methods (LIME and SHAP) for feature selection, validating our findings with statistical significance tests and demographic fairness analyses to mitigate spurious, artifact-driven correlations. Empirical results demonstrate that an optimized subset of explainable AI (XAI)-selected features combined with an MLP architecture achieves a state-of-the-art test accuracy of 82\%. Ultimately, this work provides a transparent framework for robust and ethical assistive technologies that can be applied to any other binary task.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：自动语音抑郁检测系统在实际临床应用中面临两大障碍：① 模型不透明（黑箱），无法提供临床可解释的决策依据；② 存在人口统计学偏差（如性别、年龄组间性能差异），可能加剧不公平。
- **整体含义**：现有研究多依赖自监督学习（SSL）的深度嵌入或复杂深度学习模型，虽在基准上取得高准确率（如>90%），但往往存在说话人泄露、利用访谈者提示等虚假相关性，且缺乏对可解释性和公平性的系统评估。本文旨在构建一个**兼顾可解释性与性能**的透明框架，使用低复杂度模型（RF、SVM、MLP）和人类可理解的声学特征，并通过严格的统计检验和公平性分析验证鲁棒性，以推动临床可信的辅助诊断技术。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 核心思想
- 采用**低复杂度机器学习模型**（RF、SVM、MLP）结合手工设计的声学特征（MFCCs、eGeMAPS），避免深度黑箱模型。
- 利用可解释性方法（LIME、SHAP）进行**特征选择**，替代常规的基于算法重要性的特征选择（如树模型的特征重要性），以过滤掉虚假、伪影驱动的特征。
- 通过**三种统计检验**（置换检验、参与者vs访谈者一致性检验、类别分离检验）评估模型是否真正捕捉到抑郁相关信号，并排除因访谈者语音、说话人泄露等导致的虚假性能。
- 引入**归一化期望成本（NEC）** 替代准确率作为核心指标，更公平地反映模型相对于随机基线（基于先验概率）的增益。
- 进行**人口公平性分析**（年龄×性别交织群体），使用NEC评估跨组表现差异。

### 关键技术细节
1. **数据集与预处理**：使用DAIC-WOZ扩展数据集（含E-DAIC新参与者），通过WhisperX进行说话人日志和转录，自动过滤非参与者语音。自定义80/20随机分割（参与者不重叠），避免说话人泄露，并确保训练/测试集声学平衡。
2. **特征提取**：
   - 参与者语音分段（>4秒）后，提取MFCCs（含一阶/二阶差分）及相关低层描述符（过零率、谱通量等），以及88维eGeMAPS特征集。
   - 时间特征：响应延迟、语速、发音率、语音/沉默时长比等。
   - 所有特征按说话人级别平均，得到每个参与者一个特征向量。
3. **特征选择策略**（五种）：
   - 基线（随机15个特征）
   - 统计选择（sigst）：基于检验p值排序（t检验/Mann-Whitney U检验）
   - 系统重要性选择（SYSsel）：基于排列重要性（交叉验证均值）
   - LIME：基于局部可解释模型全局聚合（平均绝对重要性）
   - SHAP：类似随机采样
   - 每种策略选出**15个**最重要特征，并在训练集内用分层K折交叉验证（5折×10个随机种子，50次）进行。
4. **分类模型**：
   - RF：10棵树，class_weight='balanced'
   - SVM：RBF核，C=10，γ='scale'
   - MLP：两层隐含层（每层1024个神经元），ReLU激活，Adam优化，L2正则化α=1e-4，训练500迭代
   - 输出经Platt缩放（SVM用概率化）进行校准，最终在logit空间线性融合（logit-mean融合）。
5. **评估指标**：
   - **NEC**（归一化期望成本）：设定假阴性和假阳性成本相等（cFN=cFP=1），归一化到随机基线（NEC=1表示与随机一样差，<1表示有增益）。可解释为错误率归一化。
   - 同时报告准确率（ACC）。
6. **统计测试**（三种）：
   - Test A（置换检验）：标签打乱后比较实际NEC与零分布。
   - Test B（一致性检验）：比较参与者语音与访谈者（Ellie）语音系统NEC的差异，看模型是否利用访谈者语音作为捷径。
   - Test C（类别分离检验）：比较两类概率分布的Mann-Whitney U检验。
