## 摘要
工作记忆是认知的基础过程。因此，支持工作记忆表现的神经机制在认知神经科学中备受关注。我们提供了一个公开可访问的数据集，包含123名健康成年人（平均年龄28.58岁，标准差7.56；年龄范围18-46岁；75名女性），以促进对人类工作记忆表现背后神经机制的研究。在两天的评估中，参与者完成了一系列评估工作记忆和其他认知领域（第一天）的认知任务，随后进行62通道脑电图记录（第二天）。EEG数据在静息状态、视觉工作记忆任务期间以及针对工作记忆关键脑区的单脉冲经颅磁刺激（TMS-EEG）后采集。本文提供了研究设计、方法和数据特征的详细概述。

## Abstract
Working memory is a fundamental process that underlies cognition. Accordingly, the neural mechanisms that support working memory performance are of great interest in cognitive neuroscience. We present a publicly accessible dataset comprising 123 healthy adults (mean age = 28.58 years, SD = 7.56; age range = 18-46 years; 75 females) to facilitate investigation of the neural mechanisms underlying working memory performance in humans. Across two days of assessment, participants underwent a battery of cognitive tasks assessing working memory and other cognitive domains (day 1), followed by 62-channel EEG recordings (day 2). EEG data were acquired at rest, during a visual working memory task, and following single-pulse transcranial magnetic stimulation (TMS-EEG) targeting key brain regions involved in working memory. This article provides a detailed overview of the study design, methodology, and data characteristics.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：工作记忆是支撑高阶认知（如推理、情绪调节、社会行为）的基础过程，尤其视觉工作记忆（VWM）能力差异可解释健康个体及认知障碍患者（如精神分裂症、高龄）认知能力超过40%的方差。然而，现有研究存在两大局限：一是特定任务下的EEG机制能否泛化到更广泛的工作记忆表现及其他认知领域尚不清楚，且需要大样本（通常至少100人）进行潜变量建模；二是EEG空间分辨率有限，难以将神经机制精确定位到特定皮层区域。TMS-EEG技术可弥补空间不足，但存在高个体间变异和伪影问题，亟需公开的大规模、高质量、严格实验控制的数据集以提升可重复性。

- **整体含义**：本研究旨在提供一个包含123名健康成人的多模态公开数据集（静息态EEG、任务态EEG、单脉冲TMS-EEG以及成套认知行为测试），以促进对工作记忆神经机制的深入、可重复研究，并为EEG/TMS-EEG数据处理流程优化、计算建模等提供基准资源。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：通过集成行为测试、传统EEG和TMS-EEG三种模态，结合大样本，实现“深度表型”工作记忆的神经关联。利用结构方程模型等潜变量方法可将行为测试结果与EEG指标关联；TMS-EEG可非侵入性刺激关键皮层区域（如背外侧前额叶、前运动皮层、后顶叶皮层），记录TMS诱发电位（TEPs），反映局部皮层反应性、兴奋/抑制平衡、振荡特性和功能连接。

- **关键技术细节**：
  - **EEG采集**：62通道Ag/AgCl电极，10-20系统，采样率10 kHz，参考FPz、接地AFz，皮肤阻抗<5 kΩ。使用SynAmps 2系统（Neuroscan, Compumedics）。
  - **TMS-EEG**：MagVenture CB60八字形线圈，双相脉冲（前后-后前方向）。利用立体神经导航系统基于个体MRI引导线圈位置。刺激强度根据静息运动阈值（RMT）经线圈-皮层距离校正后确定（前94人用100% RMT_adj，后29人用120% RMT_adj）。刺激部位：左半球背外侧前额叶（BA9）、前运动皮层（BA6）、后顶叶皮层（BA7），以及肩部控制条件。每块120个脉冲，随机间隔4-6秒。使用泡沫片减少振动，白噪声掩蔽TMS声音。TMS-EEG数据预处理：去除伪影（-1~5 ms插值），降采样至1000 Hz，独立成分分析（ICA）去除肌肉和眨眼伪影等。
  - **行为测试**：涵盖工作记忆存储容量（变化检测任务：Bar Task、Colour Task；连续回忆任务：Orientation Task）、工作记忆容量伴干扰（复杂广度任务：Symmetry Span、Operation Span、Backwards Digit Span）、工作记忆更新（2-back任务）、加工速度（Symbol Search）、抑制控制（Go/No-Go、Stop-Signal Task、Simon Task）。
  - **EEG数据预处理**：对公开数据使用RELAX流水线（带通0.25-80 Hz，陷波47-53 Hz，PREP去噪通道，多通道Wiener滤波，ICLabel ICA分类及小波增强ICA去伪影，球面插值，平均重参考）。
