## 摘要
将脑信号翻译成文本可以恢复严重瘫痪患者的沟通能力，但迄今为止实用的系统依赖于侵入性的皮层脑电图（ECoG）。脑电图（EEG）提供了一种非侵入性的替代方案，并且从EEG到文本（EEG2Text）已被广泛探索。然而有趣的是，EEG2Text模型通常依赖于教师强制评估；没有它，模型无法生成有意义的解码。这种依赖性阻碍了EEG2Text在现实非学术环境中的应用。这引发了许多关于EEG2Text是否有意义的方向的争论，以及EEG是否真正包含可解码的语言信息。这里，通过采用神经心理学启发的范式，我们发现现有的EEG2Text基准忽略了EEG的不稳定性，这一缺陷混淆了推论并引发了争论。我们的实验为无教师强制EEG2Text解码的可行性提供了关键证据。因此，我们使用128通道高密度EEG帽组装了EEG到文本语料库（COFETT），提供了一个专门评估EEG2Text模型的基准。与多个现有基准相比，COFETT在区分模型性能方面达到了最先进水平，并实现了稳健的、无教师强制评估，从而为实用EEG2Text应用开辟了道路。COFETT在https://github.com/baoyudu/COFETT开源。

## Abstract
Translating brain signals into text could restore communication for people with severe paralysis, yet practically usable systems to date rely on invasive electrocorticography (ECoG). Electroencephalography (EEG) offers a non-invasive alternative, and EEG-to-text (EEG2Text) has been widely explored. Interestingly, however, EEG2Text models generally rely on teacher-forcing evaluation; without it, they fail to generate meaningful decoding. This reliance prevents EEG2Text from being applied in real-world, non-academic settings. This has fueled numerous debates about whether EEG2Text is a meaningful direction, by extension, and whether EEG truly contains decodable linguistic information. Here, using a neuropsychology-informed paradigm, we find that existing EEG2Text benchmarks have neglected EEG instability, a flaw that has confounded inference and sparked debate. Our experiments furnish key evidence for the feasibility of teacher-forcing-free EEG2Text decoding. Accordingly, we assemble the Corpus OF Eeg-To-Text (COFETT) using a 128-channel high-density EEG cap, providing a benchmark dedicated to evaluating EEG2Text models. In comparisons with multiple existing benchmarks, COFETT achieves SOTA ability to distinguish among model performances and enables robust, teacher-forcing-free evaluation, thereby opening a path toward practical EEG2Text applications. COFETT is open sourced in https://github.com/baoyudu/COFETT.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：当前 EEG-to-Text（EEG2Text）系统普遍使用教师强制（teacher-forcing）进行评估——即在解码时输入真实的前一个 token 来预测下一个 token。这种评估方式掩盖了暴露偏差（exposure bias），导致性能被夸大，且模型在无 teacher-forcing 时无法生成有意义的文本。
- **争议**：有研究表明，在 teacher-forcing 评估下，将 EEG 输入替换为随机噪声后，模型性能几乎无变化，这引发了对 EEG 是否真正包含可解码语言信息的质疑，以及 EEG2Text 方向是否可行的广泛争论。
- **根源**：论文发现现有基准（如 ZuCo）忽略了 EEG 固有的不稳定性（instability）——同一被试在不同试次或 session 间信号统计分布存在显著漂移（由认知状态、电极位移、生理波动等引起）。这种不稳定性使模型无法学到稳定的语言相关特征，从而在无 teacher-forcing 解码时失败。
- **意义**：若能在考虑 EEG 不稳定性的前提下设计合理的基准并实现无 teacher-forcing 评估，则可为 EEG2Text 的实用性提供关键证据，推动非侵入式脑机接口在恢复语言沟通上的应用。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心理念**：
  1. **对抗 EEG 不稳定性**：通过让同一被试在精心间隔的时间内多次想象完全相同的句子，利用重复测量来提取 EEG 中稳定的语言相关成分，同时平均掉随时间变化的噪声。
  2. **无教师强制评估**：放弃传统 token 级指标（BLEU、ROUGE、WER），改用句子级嵌入对齐度量——计算 EEG 嵌入与文本嵌入之间的 Pearson 相关系数，从而避免依赖真实历史 token 进行解码。

- **关键技术细节**：
  1. **实验范式（图 2）**：
     - 每个被试进行 8 天实验，每天 4 个 session，每个 session 包含 m 个 trial（day1-4: m=281；day5-8: m=252）。
     - 每个 trial 分三阶段：逐词阅读 → 逐词回忆（内语音想象）→ 休息。重点分析回忆阶段的内语音信号。
     - 文本素材覆盖 39 个日常生活类别，避免自由想象引入的偏差，且阅读阶段激活语言区有助于过渡到内语音状态。

     - 训练时：使用预训练语言模型（LaBSE）的文本嵌入作为监督信号，训练 EEG 编码器使其输出嵌入与目标句子的嵌入对齐。
     - 测试时：直接将 EEG 嵌入与候选文本嵌入比较，计算 Pearson 相关系数，无需 teacher-forcing。
     - 训练/测试数据来自同一被试的不同 session，避免数据泄露。
  4. **可行性分析工具**：

## 6. 论文的主要结论与发现

1. **现有基准（ZuCo）不可靠**：在 teacher-forcing 评估下，ZuCo 上真实 EEG 与随机噪声得分无显著差异，说明其无法衡量模型是否真正学习到 EEG 中的语言信息。
2. **EEG 不稳定性是关键障碍**：忽视跨 session 信号漂移导致模型无法从重复测量中提取稳定语言特征，这是无 teacher-forcing 解码失败的根本原因。
3. **COFETT 实现了有效的无教师强制评估**：通过重复内语音想象范式和嵌入对齐度量，COFETT 能够区分不同模型的真实学习能力，噪声对照下性能显著下降，验证了方法的有效性。
4. **EEG 确实包含可解码的语言信息**：统计证据（MINDFUL/CCA）和 scaling-law 现象均支持语言信息的存在，但信号强度弱，需要大量重复数据才能有效提取。
5. **预测数据需求**：要达到类似机器翻译的性能，目前需要约 700 次重复/句子，超出当前数据集规模约 40 倍，凸显大规模数据采集的必要性。
