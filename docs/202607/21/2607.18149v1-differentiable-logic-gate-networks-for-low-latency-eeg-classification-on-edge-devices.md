## 摘要
在边缘设备上进行实时脑电（EEG）分类受限于传统神经网络的浮点运算。我们研究了可微分逻辑门网络（Diff-Logic）作为一种硬件原生的替代方案，该方案将模型编译为可通过位运算CPU执行的纯布尔电路。通过在涵盖两个分类任务（二分类痴呆检测和三分类情绪识别）的四个EEG数据集上进行严格的等参数实验，我们在四个复杂度层级（50k-500k参数）上将Diff-Logic与相同容量的多层感知机（MLP）和二值化神经网络（BNN）基线进行了比较。在痴呆筛查任务中，Diff-Logic取得了80.2%的宏F1分数，比MLP基线高出6.8%。在情绪识别任务中，MLP保持了中等性能优势，但在部署于功耗受限（7W）的Nvidia Jetson Orin Nano CPU（单核）上时，其延迟高出2.3倍，模型大小高出14倍。关键在于，当模型规模增加10倍时，Diff-Logic的推理时间几乎保持不变，在最大复杂度层级上相比MLP实现了2.9倍的峰值加速。我们的研究结果确立了基于逻辑的神经架构作为资源受限脑机接口的一种实用范式，在满足便携式边缘部署的延迟和内存约束的同时，实现了具有竞争力或更优的性能。代码已在GitHub上开源：https://github.com/Shyamal-Dharia/eeg-difflogic

## Abstract
Real-time EEG classification on edge devices is bottlenecked by the floating-point arithmetic of conventional neural networks. We investigated Differentiable Logic Gate Networks (Diff-Logic) as a hardware-native alternative that compiles models into pure Boolean circuits executable via bitwise CPU operations. Through rigorous iso-parameter experiments across four EEG datasets spanning two classification tasks, binary dementia detection and 3-class emotion recognition, we compared Diff-Logic against matched-capacity Multi-Layer Perceptron (MLP) and Binarized Neural Network (BNN) baselines at four complexity tiers (50k-500k parameters). On dementia screening, Diff-Logic achieved 80.2% Macro F1, outperforming the MLP baseline by 6.8%. On emotion recognition, the MLP retained a moderate performance advantage but incurred a 2.3$\times$ higher latency and 14$\times$ larger model size when deployed on a power-constrained (7W) Nvidia Jetson Orin Nano CPU (Single-core). Critically, Diff-Logic inference time remained nearly constant across a 10$\times$ increase in model scale, achieving a peak speedup of 2.9$\times$ over MLPs at the largest complexity tier. Our results establish logic-based neural architectures as a practical paradigm for resource-constrained brain-computer interfaces, achieving competitive or superior performance while natively satisfying the latency and memory constraints of portable edge deployment. Code is available on GitHub: https://github.com/Shyamal-Dharia/eeg-difflogic

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：在边缘设备（如便携式脑机接口系统）上实现实时脑电（EEG）分类时，传统神经网络依赖高精度的浮点乘法累加（MAC）运算，导致高延迟和高能耗，成为部署瓶颈。此外，模型常需将处理卸载到云端，引入额外延迟并带来隐私风险。
- **研究背景**：尽管深度学习在EEG解码中取得显著进展（如EEGNet、EEG-Conformer、LaBraM等大规模基础模型），但大多数方法依赖浮点运算，难以满足电池供电、低功耗边缘设备的实时性要求。现有的量化、剪枝等方法（如二值化神经网络BNN）虽减少计算，但仍保留稠密连接和累加步骤，吞吐量受限。
- **整体意义**：探索可微分逻辑门网络（Diff-Logic）作为硬件原生替代方案，通过将模型编译为纯布尔逻辑电路（仅需位运算），有望实现推理延迟几乎不随模型规模增长、存储开销极小的特性，为资源受限的脑机接口边缘部署提供可行范式。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：Diff-Logic网络由可学习的布尔逻辑门组成，训练时每个门维护一个概率分布（覆盖全部16种可能的双输入布尔函数），通过梯度下降优化门的选择。训练收敛后，每个门“硬化”为单一逻辑函数（如AND、OR、XOR等），形成稀疏、静态的布尔电路。推理时完全通过比特位运算（ALU）执行，无需浮点运算和累加寄存器。

- **关键技术细节**：
  1. **输入量化——温度计编码**：将连续EEG特征（经Min-Max归一化）映射为长度为T（T=15）的二进制向量，保留数值顺序。例如，归一化值0.6对应前k位为1、后T-k位为0的向量。输入维度扩大T倍（如95维特征→1425位二进制）。
  2. **可微分逻辑层**：每个神经元连接两个二进制输入，输出为16种可能逻辑函数的软概率加权和：
     \[
     y_{\text{soft}} = \sum_{i=1}^{16} \frac{\exp(\alpha_{n,i})}{\sum_j \exp(\alpha_{n,j})} \cdot f_i(x_1, x_2)
     \]
     其中\(\alpha_{n,i}\)为可学习参数，\(f_i\)为不同布尔函数（见原文表A3）。该形式对\(\alpha\)和输入\(x\)均可微。
  3. **硬化与推理**：训练后选择每神经元概率最大的函数，形成静态布尔电路。输出层采用分组求和（Group Sum），将最后层比特划分为C组（类别数），每类logit为组内1的个数，送入温度缩放Softmax产生概率（温度\(\tau=30\)）。
  4. **网络结构**：堆叠多个逻辑门层，层间采用随机稀疏连接。与MLP/BNN进行等参数比较，每层级神经元/门数量按参数量匹配调整（见原文表A4）。
  5. **编译与部署**：硬化后的逻辑电路可编译为C共享库，在边缘CPU上通过比特位运算执行，无需ONNX Runtime等浮点框架。

- **算法流程**：
  - 训练阶段：输入温度计编码的二进制特征 → 通过可微分逻辑层（软门） → 分组求和 → Softmax损失 → 反向传播更新\(\alpha\)。
  - 推理阶段：硬化门函数 → 编译为纯位运算C代码 → 在CPU上运行（仅需AND、OR、NOT、XOR等操作）。

## 6. 主要结论与发现

1. **分类性能**：
   - 痴呆检测任务：Diff-Logic持续优于MLP和BNN，最佳宏F1达80.2%（CN vs AD，100k参数），超过MLP约9.4个百分点。
   - 情感识别任务：MLP在中低参数层级领先（约62% vs 55-61%），但在500k参数时Diff-Logic追平（61.0% vs 61.3%）。BNN表现最差且不稳定（std高达±12%）。

2. **推理效率**：
   - Diff-Logic的推理延迟几乎不随参数规模增长：从50k到500k参数，延迟仅从0.19ms略微增至0.22ms（痴呆任务），而MLP从0.107ms线性增长至0.631ms。
   - 在最大层级500k参数时，Diff-Logic相比MLP获得2.91×加速（痴呆）和2.30×加速（情感），相比BNN加速更大。
   - 模型存储大小：Diff-Logic仅140KB（500k参数），而MLP约1959KB（14×更小）。
   - BNN尽管使用二进制，因需累加步骤（popcount），延迟甚至高于Diff-Logic。

3. **缩放特性**：Diff-Logic的延迟-参数曲线近乎平坦，因为编译后电路的计算开销由电路深度（层数）而非宽度（门数）主导。这使其在模型增大时仍保持恒定低延迟，是传统MAC架构无法达到的优势。

4. **优越的归纳偏置**：Diff-Logic自然适用于临床EEG中的阈值型决策边界（如频带功率阈值），与痴呆检测任务契合度高于连续激活的MLP。
